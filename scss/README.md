# Change Color

## Install

```
npm i
```

## Build

```
npm run build
```

## Copy CSS file

Copy `scss/dist/IMKitAppColor.css` to `static/css/IMKitAppColor.css`

## HTML

Add Class adn CSS file

```html
<link href="static/css/IMKitAppColor.css" rel="stylesheet" />

<body class="IMKitApp-color"></body>
```
